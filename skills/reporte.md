# Skill: Reportes Temáticos

## Propósito
Convertir datos técnicos en narrativas comprensibles y atractivas (Data Storytelling).

## Patrones de Diseño
1. **Layout Editorial:** Uso de columnas anchas para texto y márgenes generosos.
2. **Visualizaciones Embebidas:** Gráficos de `recharts` o mapas que se activan al hacer scroll.
3. **Citas Destacadas:** Texto en itálica grande para resaltar hallazgos clave de científicos o ministros.
4. **Descargas:** Botones de acción flotantes para obtener el PDF o la base de datos vinculada.

## Iconografía y Estética
- **Icono Principal:** `FileText`.
- **Contraste:** Fondo `white` puro con acentos en `slate-50` para separar secciones narrativas.
- **Scroll Hijacking:** Opcional, pero se prefiere scroll suave con `framer-motion` (si estuviera disponible) o CSS transitions.

## Metadatos Necesarios
- Autor Institucional (ej. CCAD).
- Fecha de Publicación.
- Ejes relacionados para navegación cruzada.
